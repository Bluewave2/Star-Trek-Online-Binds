# Star Trek Online Binds
In-game binds for Star Trek Online, an online MMORPG video game.

You can enter the binds in-game via chat by typing ```/bind "key" "command"``` or by adding them via a binds file and loading the binds file on a specific character.
# Ground

## 1. Walk bind (toggle)
```/bind "rctrl" "++walk"```

Binds **right control** to toggle walking.

## 2. Walk bind (hold)
```/bind "rctrl" "+walk"```

Binds **right control** to walk while being held.

# Ground and Space
## 1. Diagnostics (FPS, ping, memory, graphs) display
```/bind "f5" "++showfps $$ ++showmem $$ ++fpsgraph $$ ++fpshisto $$ ++netgraph $$ ++nettiminggraph"```

Binds **F5** to display all diagnostic information on the first press, and disable the diagnostic information on the second press.

## 2. Character switch
```/bind "f9" "gotoCharacterSelect"```

Binds **F9** to instantly log the current character out and display the switch characters menu.

## 3. Sound toggle
```/bind "alt+f2" "sndEnable" "sndDisable"```

Binds **ALT + F2** to two functions, **on short press** of the key combination it will enable the in-game sound, **on hold (1.5s)** it will disable the in-game sound.

## 4. Legacy character overview menu behavior
```bind "u" "GenSendMessage Root ShowInventory $$ GenSendMessage Root ShowStatus"```

Binds **U** to display the character overview menu (default behavior) and also open the inventory alongside it (legacy behavior).
